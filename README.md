# HotGaze

Predict visual attention on UI screenshots — heatmap overlays and machine-readable attention scores.

> **Work in progress** — v0.1.0 coming soon.

## Quickstart (coming)

```bash
pip install hotgaze
hotgaze run screenshot.png -o overlay.png
hotgaze score screenshot.png --region cta:100,200,300,80 --json
hotgaze compare a.png b.png --region cta:100,200,300,80
```
